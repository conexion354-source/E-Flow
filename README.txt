E-Flow Pro completo

Incluye:
- Login con clave 1234nn
- Panel por colores
- Calendario mensual
- Rechazados y anulados desplegables
- Detalle completo de cada cheque:
  fecha de emisión, proveedor, fecha de pago, banco, número de cheque, tipo, monto, estado y observaciones
- Firebase Cloud Messaging integrado
- Token del dispositivo
- Service worker para recibir push en segundo plano

Importante:
Esta versión ya puede RECIBIR push, pero para que lleguen avisos automáticos con la app cerrada todavía hace falta enviar mensajes desde Firebase Console, un backend propio o una función programada.

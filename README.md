# Panel de cheques E-flow para GitHub Pages

Esta versión ya está preparada para publicar en:

https://conexion354-source.github.io/E-Flow/

## Instalar
npm install

## Probar local
npm run dev

## Generar publicación
npm run build

Después subí el contenido de la carpeta `dist` al método de publicación que uses en GitHub Pages.
